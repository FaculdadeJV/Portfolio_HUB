> Hub central para documentação e apresentação das atividades práticas da disciplina de Desenvolvimento Web.

## 📋 Sobre o Projeto

Este portfólio foi desenvolvido para documentar e anexar todas as atividades realizadas em sala de aula na disciplina de Desenvolvimento Web. O projeto consiste em uma página hub com menu de navegação, submenus para acesso rápido às aulas e relatórios, cards interativos e integração com arquivos PDF.

### 🎯 Objetivo

Consolidar os conhecimentos fundamentais de front-end através da prática constante, criando um repositório visual da evolução técnica e servindo como portfólio demonstrativo das competências adquiridas na disciplina.

## 🚀 Funcionalidades

-  **Menu de navegação fixo** com efeito glass
-  **Submenus suspensos** para Aulas e Relatórios
-  **Cards interativos** na página inicial com efeitos hover
-  **Links para PDFs** que abrem em nova aba
-  **Design responsivo** adaptado para mobile
-  **Paleta de cores** azul e vermelho
-  **Efeitos visuais** com gradientes e animações suaves

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Descrição |

| HTML5 | Estruturação semântica do conteúdo |
