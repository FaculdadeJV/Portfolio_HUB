Portfólio de Desenvolvimento Web

- Portfólio

Este portfólio foi criado para anexar e documentar todas as atividades realizadas em sala de aula na disciplina de Desenvolvimento Web. O menu acima possui informações e visualização de todos os projetos e relatórios desenvolvidos ao longo do semestre.

- Desenvolvimento

Todos os trabalhos foram desenvolvidos seguindo um tema proposto em sala de aula, com o objetivo de aprendermos conceitos novos e adquirirmos habilidades práticas para o desenvolvimento web. Cada atividade aborda diferentes tecnologias e técnicas, desde HTML semântico até estilização avançada com CSS.

- Objetivo

Consolidar os conhecimentos fundamentais de front-end através da prática constante, criando um repositório visual da evolução técnica. Este projeto também serve como base para futuros trabalhos e como portfólio demonstrativo das competências adquiridas na disciplina.
